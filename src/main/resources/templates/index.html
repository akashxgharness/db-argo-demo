This UI is themed dynamically based on the container image tag provided to the application via your EKS configuration.

  • Development when the tag contains dev.
  • Staging when the tag contains staging.
  • Production when the tag contains prod or matches a semantic version like v1.2.3.
  • Default for any other tag.
